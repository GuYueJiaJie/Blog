[3.无重复字符的最长子串]

思路就是使用滑动窗口，设置两个指针，动态的调整滑动窗口的左右边界。

## 方法一

对方法二的优化，当查找到重复的值时，直接跳过该窗口。

```javascript
/**
 * @param {string} s
 * @return {number}
 */
var lengthOfLongestSubstring = function(s) {
  if (s.length === 0) return 0;
  let map = new Map();
  let max = -Infinity;
  for (let i = 0, j = 0; j < s.length; j++) {
    if (map.has(s[j])) {
      i = Math.max(map.get(s[j]), i);
    }
    max = Math.max(max, j - i + 1);
    map.set(s[j], j + 1);
  }
  return max;
};
```

## 方法二

使用 Set 来进行记录是否有重复值，但是

```javascript
/**
 * @param {string} s
 * @return {number}
 */
var lengthOfLongestSubstring = function(s) {
  if (s.length === 0) return 0;
  let i = 0,
    j = 0,
    set = new Set(),
    max = -Infinity;
  let len = s.length;
  while (i < len && j < len) {
    if (!set.has(s[j])) {
      set.add(s[j++]);
      max = Math.max(j - i, max);
    } else {
      set.delete(s[i++]);
    }
  }
  return max;
};
```
